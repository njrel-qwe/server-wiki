# server — вики сервера

Статический сайт-вики (HTML/CSS/JS, без сборки) в полной Minecraft-стилизации.
Описывает игровые механики сервера: фракции, войны, аванпосты, контракты, боссы,
экономику, суд, кастомные предметы, правила и ЧаВо.

## Запуск локально
Просто откройте `index.html` в браузере. Либо поднимите статический сервер:
```
python -m http.server 8765 --directory .
```
и откройте http://localhost:8765

## Хостинг на GitHub Pages
1. Создайте репозиторий и запушьте содержимое папки `server-wiki-v2` в корень ветки `main`.
2. Settings → Pages → Source: `Deploy from a branch` → ветка `main`, папка `/ (root)`.
3. Через минуту сайт будет доступен по адресу `https://<user>.github.io/<repo>/`.
Сборка не нужна — это чистая статика.

## Что заменить под себя
- **IP сервера** — в `js/site.js`, объект `CONNECT.ip` (сейчас плейсхолдер `play.server.ru`).
  Discord уже указан реальный.
- **Скриншоты** — положите 6 файлов в `assets/screenshots/` с именами:
  `spawn.png`, `mayak.png`, `contracts.png`, `court.png`, `tc.png`, `tc-inside.png`.
  Пока файла нет — на странице показывается рамка-заглушка с подсказкой имени.
  (см. `assets/screenshots/_READ_ME.txt`)

## Шрифт
Заголовки/UI используют семейство `PixelMC`:
- латиница берётся из `assets/fonts/Minecraftia.ttf` — **если вы положите этот файл**
  (по умолчанию его нет, лицензионно чистую копию можно скачать с dafont);
- всё остальное (включая кириллицу) — из `assets/fonts/Monocraft.ttf` (уже в комплекте,
  пиксельный Minecraft-стиль с полной кириллицей).
Если `Minecraftia.ttf` не добавлять — весь текст рисуется Monocraft, и это уже выглядит
аутентично. Minecraftia кириллицы НЕ содержит, поэтому она применяется только к латинице.

## Структура
```
index.html              — главная
pages/                  — страницы механик + items.html (каталог), item.html (карточка), qa.html (ЧаВо)
css/style.css           — вся стилизация (Minecraft GUI)
js/site.js              — каркас (сайдбар, шапка, навигация, футер)
js/items-data.js        — данные всех кастомных предметов
js/items.js             — рендер каталога и карточек предметов
assets/items/           — иконки предметов (из текстурпака)
assets/textures/        — текстуры блоков (фоны/панели)
assets/fonts/           — Monocraft (+ опц. Minecraftia)
assets/cursors/         — пиксельный курсор-меч
assets/screenshots/     — ваши скриншоты сервера
```

## Стилизация (кратко)
- Фон — текстура deepslate; контент — каменная панель (читаемый тёмный фон).
- Карточки/кнопки/чипы — светло-серые инвентарные панели `#C6C6C6` с пиксельным бевелом.
- Кнопки повторяют меню MC: hover → жёлтый текст, active → «вдавливание».
- Шрифты с игровой тенью (`text-shadow`), `image-rendering: pixelated`, курсор-меч.
- Анимации ступенчатые (`transition: steps()`), без плавных градиентов/скруглений.
